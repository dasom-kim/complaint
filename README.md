# 매교역 교통 민원 도우미

이 프로젝트는 매교역 인근 주민들의 인프라 개선을 위한 민원 접수를 돕는 정적 웹 애플리케이션입니다.

## 개발 환경 설정 및 실행

1.  **의존성 설치**: 프로젝트 폴더에서 다음 명령어를 실행합니다.
    ```bash
    npm install
    ```

2.  **개발 서버 실행**: 로컬에서 프로젝트를 확인하려면 다음 명령어를 사용하세요.
    ```bash
    npm start
    ```
    이 명령어는 `public` 폴더를 기준으로 로컬 서버를 실행하고, 웹 브라우저를 자동으로 열어줍니다. (`live-server` 사용)

3.  **JavaScript 번들링**:
    *   개발 중 `main.js` 파일의 변경사항을 자동으로 `bundle.js`에 반영하려면, 새 터미널을 열고 다음 명령어를 실행하세요.
        ```bash
        npm run watch
        ```
    *   배포 전에 수동으로 빌드하려면 다음 명령어를 사용하세요.
        ```bash
        npm run build
        ```

## 배포

`public` 폴더는 Firebase 호스팅 등 모든 정적 사이트 호스팅 서비스에 배포할 수 있는 모든 파일을 포함하고 있습니다.

## 구조

*   `package.json`: 프로젝트 메타데이터 및 의존성 관리
*   `public/`: 배포를 위한 모든 정적 파일을 포함하는 루트 폴더입니다.
    *   `index.html`: 메인 애플리케이션 HTML 파일
    *   `css/style.css`: 전체 스타일 시트
    *   `js/main.js`: 클라이언트 사이드 JavaScript 원본 로직
    *   `js/bundle.js`: `main.js`를 브라우저에서 실행할 수 있도록 변환한 파일 (빌드 결과물)
